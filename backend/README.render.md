# Market Fiscalía - Backend

Backend API para Market Fiscalía desplegado en Render.

## Variables de entorno requeridas en Render:

```
NODE_ENV=production
PORT=10000
DATABASE_URL=<tu_postgres_url_de_render>
JWT_SECRET=<genera_un_secret_seguro>
```

## Comandos:
- Build: `npm install`
- Start: `npm start`
